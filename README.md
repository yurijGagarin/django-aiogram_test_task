## Requirements
- python 3.9

## Installation
```bash
git clone https://github.com/yurijGagarin/django-aiogram_test_task.git
cd django-aiogram_test_task
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt or 
cp .env.example .env
# edit .env
```

## Run Django server
```bash
python manage.py runserver
```

## Run Bot

```bash
python -m bot.main 
```
